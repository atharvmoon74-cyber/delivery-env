# 🚚 Smart Delivery Optimization Environment

## Overview
This project implements a reinforcement learning environment where an agent delivers orders efficiently.

## Features
- 3 difficulty levels: Easy, Medium, Hard
- Reward-based system for optimized delivery
- Fully compliant with OpenEnv specifications

## Actions
- 0 → Move Right
- 1 → Move Left
- 2 → Deliver Order

## Observations
- Agent Position
- Pending Orders
- Steps Left
- Total Orders

## Setup

```bash
pip install -r requirements.txt
python inference.py
